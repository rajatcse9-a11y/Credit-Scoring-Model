# 💳 Credit Scoring Prediction Model

## 📌 Project Overview
This project predicts whether a loan applicant is a good or bad credit risk using Machine Learning.

The goal is to help financial institutions reduce risk by making data-driven decisions.

---

## 📊 Dataset Information
Dataset used: German Credit Dataset  

### Features include:
- Age  
- Job  
- Housing  
- Credit Amount  
- Duration  
- Purpose  
- Saving Accounts  
- Checking Account Status  
- Employment  

Target Variable:
- Good Credit (0)
- Bad Credit (1)

---

## 🔍 Exploratory Data Analysis (EDA)
- Checked missing values  
- Visualized distribution of numerical features  
- Analyzed correlation between variables  
- Studied class imbalance  

---

## 🤖 Machine Learning Model Used
- Logistic Regression  

Why Logistic Regression?
- Works well for binary classification
- Interpretable model
- Good baseline performance

---

## 📈 Model Performance
- Accuracy: 98%  
- Evaluated using train-test split  
- Confusion Matrix used for performance validation
